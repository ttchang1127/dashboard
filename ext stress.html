線形情境快選 — 橋墩剪力試算

一鍵套用預設外置PT線形，切到分頁6-4 讀 φVn 改善（P 固定 500t，僅調線形）。墩區 Vp = P·(278−墩錨y)/(100·(56.5−轉向塊2))。

四情境沿「把外置PT垂直分力 Vp 往橋墩端集中」由保守到激進遞進（跨徑 L=56.5m、梁高 278cm、錨碇 y 自頂部起算、轉向塊貼梁底）。Vp 數值為墩區每腹板（P=500t、3 腹板）：

現況（塊 L/4·3L/4，錨 100）：標準對稱線形，錨在形心≈純軸壓、底部下垂利跨中正彎矩；墩區 Vp≈21。→ 基準／跨中主控、橋墩剪力尚有餘裕時。

A 陡升段（塊2→0.85L，錨 100）：僅轉向塊2 往墩移，墩端斜段變短變陡 → Vp≈35。→ 橋墩剪力只差一點、想用最小幾何調整補足。

B 陡升+抬墩錨（塊2→0.85L，錨 60）：A 再加墩端錨碇抬至距頂 60cm，縮短斜段＋加大高差雙管齊下 → Vp≈43，兼增墩端負彎矩偏心。→ A 不夠且墩端錨碇可上移時。

C 非對稱最大化（塊2→墩前5m，錨 60）：把 Vp 榨到幾何極限（斜段僅 5m）→ Vp≈73（約現況 3.5 倍）。副作用：塊1↔塊2 間形成 Vp=0 死區（N23~N27 反無 Vp），過陡須查轉向塊最小曲率半徑。→ 缺口大／探外置PT天花板；若連 C 都 NG，代表須改斷面補強（見 6-4 腹板加厚）。

* 僅調線形（轉向塊位置＋錨碇高度），力量 P 固定 500t；P 大小請於上方「外置預力力量 P」另調（Vp 與 P 成正比）。

◆ 次彎矩 M₂ / 次剪力 V₂（內置預力）
  • 成因：本跨為連續梁端跨，以 propped cantilever（橋台端鉸支 N1、橋墩端固接 N30）模型計算，屬 1 次超靜定。內置預力使梁起拱，受支承約束而在斷面產生額外彎矩，即次彎矩 M₂（次剪力 V₂ = dM₂/dx）。
  • 力法（贅餘力 = 橋台反力，虛擬彎矩圖 m(x)=x，均勻 EI 上下相消）：
    　X₁ = − [ ∫ M₁(x)·x dx ] ÷ [ ∫ x² dx ]　（沿全跨積分，0→L）
    　M₂(x) = X₁ · x　（沿跨線性：橋台端 N1=0、橋墩端 N30 最大）；　V₂ = dM₂/dx = X₁（沿跨常數）
    　主彎矩 M₁(x) = Σ(P · e)（內置預力 × 偏心，即逐節點 Meff = [Pbot·(Ybot−12.5) − Ptop·(Ytop−12.5)]/100）；積分以梯形法沿實際節點位置計算。
  • 納入需求（sign-aware，台灣／AASHTO §3.4.3 係數 1.0）：M₂ 與載重彎矩同號（不利）才計入，反號（有利）不採信（欄位灰字）；剪力取 max(|V載重|, |V載重+V₂|)。
    　所需彎矩(含M₂) / 所需剪力(含V₂) 即為 6-3／6-4 實際採用的檢核基準。
  • 依據：橋梁知識庫 公式卡_外置預力設計 O1 §7、算例_連續梁次彎矩與服務性應力驗算、算例_端跨propped cantilever次彎矩力法；M₂=M_total−M₁。
  • ⚠️ 此處僅內置預力 M₂（補強前）；外置預力 M₂ 於 6-4（補強後）另計。橋墩端固接為近似（2 等跨精確、3 跨以上略偏保守）。

彎矩強度檢核 (φMn ≥ Mu)

◆ 計算公式與規範出處
驗核條件： φMn ≥ Mu
設計彎矩： Mu = DLF × MDL + LLF × MLL
強度折減： φ = 0.90（台灣 §8.14 一般預力彎矩；同 AASHTO 非黏結/外置PT φf=0.90）
標稱強度： Mn 由等效矩形應力塊預先計算（查表 STRESS_BLOCK_DATA）
強度比： CR = φMn / Mu（典型值 1.5～1.9；< 1.2 需重審 PT 配置）
最小強度下限②（ACI 318-19 §9.6.2）： φMn ≥ min(1.33Mu, 1.2Mcr)
Mcr = Sbot × (fr + fcpe)　[kgf·cm] ÷ 100000 → [tf·m]
fr = 2.0√f'c (kgf/cm²) = 7.5√f'c (psi) = 0.62√f'c (MPa, ACI)　≈ 40.4 kgf/cm²（註：AASHTO LRFD 採 0.97√f'c MPa，較高）
fcpe = feff,bot = 預力產生之底纖維壓應力（含偏心彎矩效應）

◆ 橋墩負彎矩 Mn 算法（Method A — M1 公式卡 fps k 法）
  • 為何不用有效預力：極限狀態時黏結鋼腱應力由有效應力 fpe 升至接近極限 fpu，故抵抗拉力應為 Aps·fps（非 Ptop=Aps·fpe）。原「fps=fpe」為 AASHTO §5.7.3.1.3b 容許的保守簡化，會低估約 30~40%、使橋墩誤判 NG。
  • 公式（壓力區在底版，寬 bbot；黏結鋼腱 k 法）：
    　Aps = Ptop ÷ fpe（由力反推面積，fpe 同 6-2 之 13000 kgf/cm²）
    　k = 2(1.04 − fpy/fpu) = 0.28（低鬆弛）；β₁ = 0.85−0.05(f'c−28)/7（f'c=40→0.764）；dp = h − 12.5
    　c = Aps·fpu ÷ (0.85f'c·β₁·bbot + k·Aps·fpu/dp)；　fps = fpu(1 − k·c/dp)；　a = β₁·c
    　Mn = Aps·fps·(dp − a/2)
  • 一致性：與正彎矩（STRESS_BLOCK_DATA 本即採 fps）同基準；Aps 來源即 6-2「有效鋼腱面積」欄（=P/13000）。輸入 fpu、fpe 於上方參數區可調。
  • 依據：橋梁知識庫 公式卡_極限強度彎曲設計 M1、AASHTO LRFD §5.7.3；橋墩列已顯示中間值 Aps/fps/c/a 供驗算。
  • ⚠️ 採實際 fps 後橋墩餘裕較小（CR 接近 1.0）為真實結果，非不安全；準確度取決於 fpe、fpu 與黏結假設正確。6-4 補強後沿用同一 Mn。
